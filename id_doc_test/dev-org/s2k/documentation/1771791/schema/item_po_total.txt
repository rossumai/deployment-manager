### PO Total (Calculated) (item_po_total)

* **Description**: This field calculates the total purchase order amount by multiplying the quantity by the unit price. It first retrieves the quantity from the `item_quantity_match` field (defaulting to 0 if not available) and the unit price from the `item_amount_base_match` field (defaulting to "0" if not available). For the price, it converts any comma decimal separators to periods before performing the calculation.

* **Field type**: formula

* **Data type**: number

* **Required**: False

* **Exported**: False

* **Hidden**: True
