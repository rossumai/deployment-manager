### Flag Virtual Discount Line (flag_virtual_discount_line)

* **Description**: This field is used to identify whether a line item represents a virtual discount. It's a hidden field that helps the system track discount lines that are not physically present on the invoice but are calculated or generated by the system.

* **Field type**: data

* **Data type**: string

* **Required**: False

* **Exported**: False

* **Hidden**: True

**Source Hook ID**: 684928
