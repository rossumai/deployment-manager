### Subtotal (Calculated) (subtotal)

- **Description**: This field calculates the sum of all line items that have a matching purchase order ID. It works by finding all line items with a valid order ID match, taking their total amounts, and adding them together. If no line items with matching purchase orders are found, it displays a warning message and returns zero. All values are rounded to 2 decimal places.

- **Field type**: formula

- **Data type**: number

- **Required**: False

- **Exported**: False

- **Hidden**: True
