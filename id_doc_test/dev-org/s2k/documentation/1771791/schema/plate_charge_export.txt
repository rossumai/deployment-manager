### plate_charge_export (plate_charge_export)

* **Description**: This field calculates a plate charge value for export purposes. It first checks if `plate_charge` has a value - if it does, it uses that value (defaulting to 0 if null). If `plate_charge` is not set, it uses the value from `plate_charge_from_lines` (also defaulting to 0 if null).

* **Field type**: formula

* **Data type**: number

* **Required**: false

* **Exported**: false

* **Hidden**: true
