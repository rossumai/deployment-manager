### backhaul_allowance_export (backhaul_allowance_export)

* **Description**: This field calculates the negative value of the backhaul allowance. It first checks if `backhaul_allowance` is set - if so, it uses that value. Otherwise, it falls back to `backhaul_allowance_from_lines`. The result is then converted to a negative number by applying a negative sign to the absolute value.

* **Field type**: formula

* **Data type**: number

* **Required**: false

* **Exported**: false

* **Hidden**: true
