### GL Code (Extension) (item_gl_code_extension)

- **Description:** This field stores the General Ledger (GL) code extension data that is not directly captured from the invoice document. It is used by an extension to store additional GL code information that may be required for accounting purposes.

- **Field type:** data

- **Data type:** string

- **Required:** false

- **Exported:** false

- **Hidden:** true
