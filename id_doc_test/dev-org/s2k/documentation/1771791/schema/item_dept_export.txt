### Dept (Export) (item_dept_export)

* **Description**: This field displays the department information from either the "item_dept_match" field if it has a value, or falls back to using the "item_dept_extension" field if the first one is not set. It essentially prioritizes the matched department value but has a backup source.

* **Field type**: formula

* **Data type**: string

* **Required**: false

* **Exported**: false

* **Hidden**: true
