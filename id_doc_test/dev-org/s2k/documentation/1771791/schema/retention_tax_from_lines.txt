### retention_tax_from_lines (retention_tax_from_lines)

* **Description**: A numerical field that stores the sum of retention tax values calculated from invoice line items. This field is not directly visible to users and is automatically populated by the system rather than being manually entered or predicted by AI.

* **Field type**: data

* **Data type**: number

* **Required**: No

* **Exported**: No

* **Hidden**: Yes

**Source**: This field is populated by hook ID 712691.
