### Check Negative Quantity (item_check_negative_quantity)

* **Description**: This field checks if the item quantity is negative. If the quantity is less than zero, it will display an error message saying "Quantity cannot be negative". This is a validation check to ensure that quantities are always positive values.

* **Field type**: formula

* **Data type**: string

* **Required**: false

* **Exported**: false

* **Hidden**: true
