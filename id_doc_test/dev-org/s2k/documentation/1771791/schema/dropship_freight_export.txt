### dropship_freight_export (dropship_freight_export)

* **description**: This field calculates the freight cost for dropship orders. If the order is flagged as a dropship (order_dropship_flag_match is "D") and a freight value exists, it uses that freight value. Otherwise, it uses the value from dropship_freight_from_lines. If either source is not available, it defaults to 0.
* **field type**: formula
* **data type**: number
* **required**: false
* **exported**: false
* **hidden**: true
