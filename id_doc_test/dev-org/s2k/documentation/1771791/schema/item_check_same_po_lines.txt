### Check Same PO lines (item_check_same_po_lines)

* **Description**: This formula field checks if the same Purchase Order (PO) line number appears in multiple invoice line items. If duplicates are found, it displays a warning message listing the line item positions that share the same PO line number and blocks automation for the affected field. This helps prevent duplicate processing of the same PO line across multiple invoice lines.

* **Field type**: formula

* **Data type**: string

* **Required**: false

* **Exported**: false

* **Hidden**: true
