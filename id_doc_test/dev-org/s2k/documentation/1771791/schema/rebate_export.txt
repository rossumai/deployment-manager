### rebate_export (rebate_export)

* **Description**: This field calculates a negative value of the rebate amount. It first checks if a rebate value is set; if so, it uses that value (defaulting to 0 if null). Otherwise, it uses the rebate_from_lines value (defaulting to 0 if null). The final value is converted to a negative number by applying a negative sign to the absolute value.

* **Field type**: formula

* **Data type**: number

* **Required**: False

* **Exported**: False

* **Hidden**: True
