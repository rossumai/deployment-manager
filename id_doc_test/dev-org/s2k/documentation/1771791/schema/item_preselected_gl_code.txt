### Preselected GL (item_preselected_gl_code)

* **Description**: This field automatically determines a GL (General Ledger) code based on specific conditions. If an item order ID match exists, it assigns the code '201000'. If an item GL by description match exists, it uses that value. Otherwise, it remains empty.

* **Field type**: formula

* **Data type**: string

* **Required**: false

* **Exported**: false

* **Hidden**: true
