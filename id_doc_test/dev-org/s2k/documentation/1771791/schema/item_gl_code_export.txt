### GL Code (Export) (item_gl_code_export)

* **Description**: This field extracts the first part of the GL code by splitting the value from "Item GL Code Match" at the underscore character and taking the first segment. For example, if "Item GL Code Match" contains "1234_ABC", this field will display "1234". If "Item GL Code Match" is not set, this field will be empty.

* **Field type**: formula

* **Data type**: string

* **Required**: false

* **Exported**: false

* **Hidden**: true
