### Discount Term Flag (discount_term_flag)

* **Description**: This field automatically determines the type of payment term based on vendor discount information. It returns "DISCOUNT TERM" if a vendor discount exists (greater than 0), "NET TERM" if no discount exists but vendor terms are specified, or "NO TERM MATCH" if neither condition is met.

* **Field type**: formula

* **Data type**: string

* **Required**: false

* **Exported**: false

* **Hidden**: true
