### Check Total Amount vs Line Item Totals (check_total_vs_line_item_totals)

* **Description**: This formula field compares the total amount due on the document with the calculated sum of all line items. If there's a difference greater than $0.01, it displays a warning message and blocks automation. The message shows the subtotal, additional charges, calculated total, and the header total amount to help identify the discrepancy.

* **Field type**: formula

* **Data type**: string

* **Required**: false

* **Exported**: false

* **Hidden**: true
