### Inventory Quantity Variances (inventory_quantity_variances)

* **Description**: This field stores numerical values representing inventory quantity variances. It tracks differences between expected and actual inventory quantities. This is a hidden field used for internal calculations and is not directly visible to users in the interface.

* **Field type**: data

* **Data type**: number

* **Required**: No

* **Exported**: No

* **Hidden**: Yes

**Source Hook**: This field is populated by hook ID 718139.
