### Use Ratio Flag (item_use_ratio)

* **Description**: This field checks if there's a mismatch between quantities while the total amounts match. It returns 'true' when all these conditions are met: the item amount total exists and equals the item PO total, AND the item quantity exists and does not equal the item quantity match.

* **Field type**: formula

* **Data type**: string

* **Required**: false

* **Exported**: false

* **Hidden**: true
