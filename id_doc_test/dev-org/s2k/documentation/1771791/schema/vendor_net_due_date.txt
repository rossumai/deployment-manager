### Vendor Due Date for Net (vendor_net_due_date)

* **Description**: Calculates the due date for vendor payment based on different scenarios:
  * If no issue date is provided, the field remains empty
  * If a specific day of month is set for payment (vendor_net_day_of_month), it calculates the due date based on:
    * The specified day of the month
    * Whether to use the current month ('C') or next month for payment
    * Adjusts for months with fewer days than specified (e.g., if day 31 is specified for February)
  * Otherwise, it adds the net payment period (number of days) to the issue date to determine the due date

* **Field type**: formula
* **Data type**: date
* **Required**: false
* **Exported**: false
* **Hidden**: true
