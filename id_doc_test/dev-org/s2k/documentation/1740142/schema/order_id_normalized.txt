### PO Number (Normalized) (order_id_normalized)

- **Description**: This field extracts and normalizes a purchase order (PO) number by taking only the numeric digits from either the manually entered order ID or the automatically detected order ID, then keeping just the last 7 digits. If no digits are found, it returns an empty string. This helps standardize PO numbers for easier matching and reference.

- **Field type**: formula

- **Data type**: string

- **Required**: false

- **Exported**: false

- **Hidden**: true
